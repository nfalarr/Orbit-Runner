# Orbit Runner 💫

Orbit Runner adalah game arcade survival berbasis HTML Canvas. Pemain mengendalikan pesawat di orbit, menghindari meteor, menembak ancaman, mengambil kristal, dan mengumpulkan power-up untuk bertahan selama mungkin.

## Fitur

- Kontrol pesawat 4 arah: kiri, kanan, maju, dan mundur.
- Sistem tembak dengan peluru ganda untuk menghancurkan meteor dan drone.
- Power-up shield, magnet, slow field, dan boost.
- Misi bertahap, level scaling, combo, skor, jarak, dan rekor lokal.
- UI responsif dengan kontrol mobile.
- Audio arcade sederhana yang bisa dimatikan.

## Cara Main

Buka `index.html` di browser modern.

Kontrol desktop:

- `A` / `ArrowLeft`: bergerak ke kiri
- `D` / `ArrowRight`: bergerak ke kanan
- `W` / `ArrowUp`: maju
- `S` / `ArrowDown`: mundur
- `F` / `Enter`: tembak
- `Space` / `Shift`: boost
- `P` / `Escape`: pause

Kontrol mobile:

- Gunakan tombol `LEFT`, `RIGHT`, `UP`, `DOWN`, `BOOST`, dan `FIRE`.
- Bisa juga drag langsung di area game untuk mengarahkan pesawat.

## Deploy

Game ini static site, jadi bisa langsung dipublish lewat GitHub Pages, Netlify, Vercel, atau hosting static lain tanpa build step.
